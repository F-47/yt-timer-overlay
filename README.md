# YouTube Timer Overlay

A Chrome extension that adds a persistent playback timer overlay to YouTube videos.

## Features

- Shows elapsed or remaining time directly on the video
- Two display styles: **pill** (text badge) and **ring** (circular progress)
- Auto-hides when the player controls are visible
- Works on all YouTube videos including fullscreen

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Y` | Toggle overlay on / off |
| `Shift + Y` | Cycle through styles and modes |

## Style Cycle

Pressing `Shift + Y` cycles through 4 styles:

1. Pill — elapsed
2. Pill — remaining
3. Ring — elapsed
4. Ring — remaining

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked** and select the project folder
